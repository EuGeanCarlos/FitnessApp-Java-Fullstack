# 🚀 FitnessApp: Java de 0 a 100 em 6 Horas

Este projeto é um **Web App (CLI Initial) para Controle de Treino e Dieta**, desenvolvido como um desafio de imersão total em Java. O objetivo é consolidar quatro pilares fundamentais da linguagem através de um projeto prático e funcional.

---

## 📚 Plano de Estudos e Revisão

Este repositório documenta a jornada de aprendizado dividida em 4 módulos principais. Se estás a usar este projeto para estudar, aqui está o roteiro:

### 1. Java: Criando a sua primeira aplicação (8h de conteúdo resumidas)
- **Foco:** Sintaxe, variáveis, entrada de dados (`Scanner`) e operadores matemáticos.
- **No Projeto:** Implementação da lógica de IMC e estrutura inicial do programa.

### 2. Aplicando a Orientação a Objetos (10h de conteúdo resumidas)
- **Foco:** Classes, Atributos, Métodos e Instanciação de objetos.
- **No Projeto:** Criação das entidades `Usuario`, `Alimento` e `Exercicio`.

### 3. Trabalhando com Listas e Coleções (8h de conteúdo resumidas)
- **Foco:** `ArrayList`, loops (`for`, `while`) e manipulação de conjuntos de dados.
- **No Projeto:** Criação do Diário de Dieta (lista de alimentos consumidos).

### 4. Consumindo API, Arquivos e Erros (10h de conteúdo resumidas)
- **Foco:** `HttpClient`, `FileWriter` e tratamento de exceções (`try/catch`).
- **No Projeto:** Salvar o progresso em um arquivo `.txt` e (em breve) consultar tabelas nutricionais externas.

---
